# 📩 Contact Form

A clean and responsive **Contact Form UI** built using **HTML and CSS**.
This project focuses on creating a visually appealing form with a gradient background, centered layout, and interactive button effects.

## 📌 Features

* Responsive centered contact form
* Gradient background design
* Transparent glass-style container
* Required field validation
* Email input support
* Styled textarea for feedback
* Hover effect on submit button

## 🛠 Technologies Used

* HTML5
* CSS3

## 🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your browser.
